# MonitoringAgent

Script to install monitoring agent with Node exporter & promtail

## PACKAGE VERSION
- node_exporter-1.3.1
- promtail : latest

Gitclone
```
git clone https://github.com/mahbubi/monitoring_agent.git
```
Add permission +x
```
chmod +x installer.sh
```
run script.
```
./installer.sh
```
